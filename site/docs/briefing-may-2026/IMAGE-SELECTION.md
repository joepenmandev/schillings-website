# Image Selection

This document governs editorial evidence curation for pages 2-5.  
It defines what imagery must communicate commercially and emotionally.  
It does not define coordinates, final layouts, or typography.

## Page 2 — Practitioner authority

**What the imagery must prove emotionally/commercially**  
The imagery must reinforce practitioner credibility, independent validation, and discretion at first review.

**What must be visible**  
Signals of practitioner standing, contextual trust cues, and a clear sense of professional authority.

**What should be minimized**  
Generic page furniture, decorative interface regions, and any non-essential navigation context.

**How browser/UI noise should disappear**  
Exclude browser chrome and reduce interface framing so evidence reads as institutional context, not web capture.

**Image dominance intent**  
One dominant evidence plate should carry the credibility signal with immediate legibility.

**Archival inset behavior**  
Use one restrained archival inset only as context contrast, subordinate in scale and visual weight.

**How imagery supports the target realization**  
It should make clear that practitioner representation influences whether serious clients and introducers proceed.

## Page 3 — Expertise clarity

**What the imagery must prove emotionally/commercially**  
The imagery must show that expertise framing is structured, legible, and commercially relevant.

**What must be visible**  
Clear capability boundaries, ownership cues, and coherent hierarchy at first scan.

**What should be minimized**  
Dense interface detail, long-form page breadth, and visual elements that dilute scope clarity.

**How browser/UI noise should disappear**  
Suppress page-shell clutter so the visual emphasis remains on capability structure and decision clarity.

**Image dominance intent**  
A dominant plate should communicate order and scope quickly, with minimal interpretive burden.

**Archival inset behavior**  
One archival inset may be used to frame prior ambiguity, kept visibly secondary and quiet.

**How imagery supports the target realization**  
It should support the belief that clearer expertise representation improves enquiry relevance and quality.

## Page 4 — Institutional authority before contact

**What the imagery must prove emotionally/commercially**  
The imagery must show that authority is inferred before engagement through composure and consistency.

**What must be visible**  
Signals of institutional steadiness, controlled presentation, and coherent route-to-contact confidence.

**What should be minimized**  
Interface novelty, feature-like emphasis, and anything that reads as design performance.

**How browser/UI noise should disappear**  
Remove elements that make the evidence read as a product walkthrough rather than institutional representation.

**Image dominance intent**  
The dominant plate should communicate pre-contact confidence with calm visual authority.

**Archival inset behavior**  
If used, the archival inset should function only as quiet context and should not compete with the primary plate.

**How imagery supports the target realization**  
It should support the realization that pre-contact authority materially affects referral confidence and mandate suitability.

## Page 5 — Operational clarity and institutional close

**What the imagery must prove emotionally/commercially**  
The imagery must show practical clarity of access while preserving institutional restraint.

**What must be visible**  
Clear directional cues, composure in next-step pathways, and confidence in progression without urgency.

**What should be minimized**  
Transactional cues, operational clutter, and visual behavior that feels process-led.

**How browser/UI noise should disappear**  
Reduce utility-layer noise so the page reads as deliberate institutional guidance, not interface instruction.

**Image dominance intent**  
Image weight should remain controlled; evidence supports closure rather than overtaking the close.

**Archival inset behavior**  
Use archival inset only if it clarifies progression; otherwise omit to preserve a calm final cadence.

**How imagery supports the target realization**  
It should support the belief that operational clarity strengthens growth quality while preserving premium positioning.
