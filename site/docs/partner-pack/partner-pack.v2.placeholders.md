## Asset Quality Rules

- Export at 2x target placement resolution.
- Prefer PNG to avoid artifacts in print.
- Keep browser chrome out of crops.
- Keep contrast moderate (editorial, not demo-like).

## Capture Validation Policy

Any screenshot used in the pack must fail validation if it contains:

- "Cannot find module"
- "Stack Trace"
- "404 Page Not Found"
- Astro/Vite overlay
- runtime error UI
- blank page
- loading skeleton
