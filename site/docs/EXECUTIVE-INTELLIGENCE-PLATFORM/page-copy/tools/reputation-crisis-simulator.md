## What it answers

- If this scenario lands tonight, what does the next 72 hours look like?
- Which stakeholders are likely to react first?
- Where are the highest-leverage intervention points?
- Which initial response priorities should come first?

---

## What it produces

A **Crisis Escalation Forecast** containing:

- Likely first headlines.
- A windowed 72-hour timeline (0-2h, 2-6h, 6-24h, 24-48h, 48-72h).
- Investor and board concern analysis.
- Media escalation speed estimate.
- Stakeholder response map.
- Reputational severity index with confidence.
- Ranked intervention leverage points.

---

## Inputs

- Scenario description in plain language.
- Optional: principal identity.
- Optional: organisation context (sector, listed/private status, jurisdictions).
- Optional: current response posture.

Inputs are not retained beyond the session by default.

---

## What you receive

A unified Schillings Intelligence Report containing:

- Top-line 72-hour synthesis.
- Headline preview.
- Timeline panel.
- Stakeholder response map.
- Reputational severity index.
- Recommended first-response priorities.
- Confidence panel.
- AI provenance disclosure.
- Linked insights to Narrative and Exposure assessments.
- Analyst review CTA.

## How it connects

- → Run the **Headline Risk Assessment** for draft communications inside the scenario.
- → Run the **Narrative Warfare Analyzer** for downstream framing pathways beyond the first cycle.
- → Run the **Executive Exposure Audit** for principal posture context.

---

## Privacy and refusal

- Inputs are not retained beyond the session by default.
- The Assessment refuses scenarios targeting named individuals with assertions of wrongdoing.
- The Assessment refuses scenarios involving minors.
- The Assessment refuses requests that imply retaliation or interference with legal or regulatory processes.
