# Live Reading Review Template

Purpose: fast behavioral review of live profile confidence.  
Target: ~10 minutes per profile.  
Mode: structured, comparable, non-aesthetic.

Reviewer lens: evaluate as someone deciding whether to risk reputation by referring a serious matter (GC, board member, journalist, adviser, family office, UHNW intermediary, litigation referrer).

## Scoring Key

- **2** = yes / strong
- **1** = partial / unclear
- **0** = no / weak
