# Editorial topic selection (Phase 3A)

**Classification:** REVIEW REQUIRED (editorial, practice leads, brand)  
**Purpose:** **How** Intelligence topics are chosen — strategic relevance and authority value — **not** a content calendar machine or SEO keyword queue.

**Related:** [`SEO-EDITORIAL-QUALITY-STANDARD.md`](SEO-EDITORIAL-QUALITY-STANDARD.md), [`SEO-EDITORIAL-AUTHORITY-STRATEGY.md`](SEO-EDITORIAL-AUTHORITY-STRATEGY.md) (category → expertise map).

---

## Core principle

**A few excellent pieces outperform many average ones** — especially in high-trust legal, reputation, and intelligence spaces.

Prefer **selective, deliberate** coverage over broad-topic saturation.

---

## For each proposed article, evaluate

Score informally (e.g. low / medium / high); **reject** if several dimensions are weak.

| Criterion | Question |
|-----------|----------|
| **Strategic relevance** | Does this advance how Schillings wants to be *known* — not just “in the news”? |
| **Authority value** | Could this earn respect, citation, or serious sharing — not only clicks? |
| **Uniqueness** | Is there a defensible angle others are not already saying better? |
| **Long-tail authority support** | Does it deepen a theme where the firm should *own* interpretive space? |
| **Reputational sophistication** | Does it elevate perception of judgment — or read as generic “thought leadership”? |
| **Cross-border relevance** | Where relevant, does it reflect jurisdictional and audience complexity? |
| **Likely citation value** | Would a journalist, academic, or serious outlet plausibly reference it? |
| **Expertise reinforcement value** | Does it **support** an expertise hub’s territory without competing commercially? |

---

## Reject topics that are

- **Generic news summaries** without Schillings interpretation.  
- **Commodity legal explainers** (“what is X”) with no elite-client implication.  
- **Trend-chasing** for visibility without institutional point of view.  
- **Low-differentiation** — interchangeable with peer content.  
- **Traffic-first** — justified mainly by search volume or cadence.  
- **Weakly aligned** to expertise ownership and Phase 2 query model (see [`SEO-QUERY-OWNERSHIP-AUDIT.md`](SEO-QUERY-OWNERSHIP-AUDIT.md)).

---

## High-value topic territories (illustrative)

Use as **prompts for commissioning**, not a coverage mandate:

- Emerging **reputational** risks and how they show up for principals and institutions.  
- **Cross-border** implications (forums, narrative spillover, enforcement climate).  
- **AI / disinformation** implications with bounded, evidence-aware claims.  
- **Media-risk** interpretation — how scrutiny behaves, not generic PR advice.  
- **Strategic consequences** of legal, regulatory, or security shifts.  
- **Elite-client implications** — proportionate, never sensational.  
- **Regulatory** shifts with practical impact on disclosure, posture, or governance.  
- **Digital threat** evolution — lawful, precise framing.

---

## Cadence philosophy

- Prefer **thematic depth** and return visits to themes over one-off noise.  
- **Sustained authority accumulation** beats reactive volume.  
- If the only reason to publish is “we haven’t posted this week”, **do not publish**.
