## How a Schillings Intelligence Report is built

A short three-step explainer rendered in the Intelligence Report UI:

1. **Signals.** Each assessment is grounded in named, documented signals — deterministic where possible, model-inferred where necessary.
2. **Methodology.** Each Index has a versioned, peer-reviewed methodology. Confidence is named alongside every score.
3. **Review.** Every output passes the AI red-line agent. Reports can be optionally reviewed and signed by a named Schillings partner.
